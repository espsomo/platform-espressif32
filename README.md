# platform-espressif32
Espressif 32: development platform for PlatformIO

# Usage

1. [Install PlatformIO](http://platformio.org)
2. Create PlatformIO project and configure a platform option in [platformio.ini](http://docs.platformio.org/page/projectconf.html) file:

### Arduino core 2.0.10 and IDF 4.4.5
Based on Arduino core 2.0.10/IDF 4.4.5 and can be used with Platformio for the ESP32, ESP32C3, ESP32S2, ESP32S3.
```       
[platformio]
platform = https://github.com/espsomo/platform-espressif32/download/2.0.10/platform-espressif32.zip
framework = arduino

```
